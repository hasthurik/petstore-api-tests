<h1> Описание проекта </h1>

Данный проект — пример автотестов для REST API сервиса Swagger Petstore (https://petstore.swagger.io).

Тесты проверяют основные CRUD-операции API и демонстрируют базовую архитектуру.

## Используемый стек
* Java 11
* JUnit 5 (Jupiter) — тестовый фреймворк
* RestAssured — HTTP-клиент для API-тестирования
* Maven — сборка и управление зависимостями
* Jackson — сериализация/десериализация JSON

## Что покрыто тестами
Тесты написаны для /pet API и включают:
* POST /pet — создание питомца
* GET /pet/{id} — получение питомца по id
* PUT /pet — обновление питомца
* DELETE /pet/{id} — удаление питомца
* GET /pet/findByStatus — поиск питомцев по статусу

# Запуск тестов:
mvn clean test
